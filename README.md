# Basic Logic Gate Design

## Project Overview
This project explains the design and working of basic logic gates used in digital electronics.

## Logic Gates Covered
- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate

## Contents
- Project Report (PDF)
- PowerPoint Presentation (PPT)
- Logic Gate Images
- Truth Tables

## Applications
Basic logic gates are used in digital circuits, computers, communication systems, calculators, and embedded systems.
